Dave Liepmann

0001-01-01

Tufte CSS provides tools to style web articles using the ideas demonstrated by Edward Tufte’s books and handouts. Tufte’s style is known for its simplicity, extensive use of sidenotes, tight integration of graphics with text, and carefully chosen typography. This demo page is a technical replicate of the original Tufte CSS page on Edward Tufte’s GitHub site. Information may not accurately reflect features of the hugo-tufte theme.
